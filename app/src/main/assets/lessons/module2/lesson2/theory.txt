<h2>Конструкция if-else</h2>
<p>Условный оператор <code>if</code> позволяет выполнять определённые блоки кода только при выполнении некоторого условия.</p>
<br>
<h3>Синтаксис</h3>
<pre><code>if условие:
    # код, который выполнится, если условие истинно
    # (обязательно с отступом)
else:
    # код, который выполнится, если условие ложно
    # (тоже с отступом)</code></pre>
<br>
<p>Пример:</p>
<pre><code>age = 18
if age >= 18:
    print("Вы совершеннолетний")
else:
    print("Вы несовершеннолетний")</code></pre>

<h3>Отступы – важны!</h3>
<p>В Python отступы определяют, какой код относится к блоку <code>if</code> или <code>else</code>. Обычно используют 4 пробела.</p>

<h3>Конструкция elif</h3>
<p>Если нужно проверить несколько условий, используется <code>elif</code> (сокращение от else if).</p>
<pre><code>mark = 85
if mark >= 90:
    grade = "Отлично"
elif mark >= 75:
    grade = "Хорошо"
elif mark >= 50:
    grade = "Удовлетворительно"
else:
    grade = "Неудовлетворительно"
print(grade)</code></pre>
