<h2>Проект: Записная книжка</h2>
<p>Создадим программу для ведения заметок, которая сохраняет их в файл.</p>

<h3>Требования</h3>
<ul>
  <li>Заметки хранятся в файле "notes.txt".</li>
  <li>Программа поддерживает команды:
    <ul>
      <li><code>add</code> – добавить новую заметку</li>
      <li><code>list</code> – показать все заметки</li>
      <li><code>delete</code> – удалить заметку по номеру</li>
      <li><code>exit</code> – выход</li>
    </ul>
  </li>
</ul>

<h3>Подсказки</h3>
<ul>
  <li>При добавлении запрашивай текст заметки и дописывай его в конец файла (режим <code>'a'</code>).</li>
  <li>Для вывода всех заметок читай файл и выводи строки с номерами.</li>
  <li>Для удаления придётся перезаписать файл без удаляемой строки.</li>
</ul>

<p>Используй <code>with</code> для работы с файлами.</p>
