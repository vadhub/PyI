<h2>Работа с файлами</h2>
<p>Программы часто должны сохранять данные между запусками. Для этого используются файлы.</p>

<h3>Открытие файла</h3>
<p>Функция <code>open(имя_файла, режим)</code> открывает файл и возвращает файловый объект.</p>
<ul>
  <li><code>'r'</code> – чтение (по умолчанию)</li>
  <li><code>'w'</code> – запись (создаёт новый файл или перезаписывает существующий)</li>
  <li><code>'a'</code> – добавление в конец</li>
</ul>

<h3>Чтение из файла</h3>
<pre><code>file = open('data.txt', 'r')
content = file.read()      # читает весь файл
print(content)
file.close()               # обязательно закрыть!</code></pre>

<p>Другие методы чтения:</p>
<ul>
  <li><code>readline()</code> – читает одну строку</li>
  <li><code>readlines()</code> – читает все строки в список</li>
</ul>
<pre><code>file = open('data.txt', 'r')
line = file.readline()
print(line)
file.close()</code></pre>
