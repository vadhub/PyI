<h2>Почему стоит установить Python на компьютер?</h2>
<p>Мобильное приложение — отличный способ начать изучение Python, но для серьёзной работы вам понадобится полноценная среда разработки на компьютере. Вот основные причины:</p>
<ul>
    <li><strong>Скорость и мощность:</strong> компьютер выполняет код быстрее, особенно сложные вычисления.</li>
    <li><strong>Работа с файлами:</strong> на компьютере вы можете создавать, читать и изменять любые файлы.</li>
    <li><strong>Бесконечные циклы и многопоточность:</strong> в мобильном приложении они ограничены, на компьютере — нет.</li>
    <li><strong>Отладка:</strong> IDE позволяют пошагово выполнять код, смотреть значения переменных, искать ошибки.</li>
    <li><strong>Внешние библиотеки:</strong> вы сможете устанавливать тысячи пакетов через pip.</li>
    <li><strong>Реальные проекты:</strong> веб-приложения, боты, анализ данных — всё это требует локальной среды.</li>
</ul>

<h2>Что можно использовать?</h2>
<p>Для написания Python-программ вам понадобятся:</p>
<ul>
    <li><strong>Интерпретатор Python</strong> — программа, которая выполняет код.</li>
    <li><strong>Текстовый редактор или IDE</strong> — где вы будете писать код.</li>
</ul>
<p>Популярные варианты:</p>
<ul>
<li><strong><a href="https://code.visualstudio.com/" target="_blank">VS Code</a></strong> — бесплатный, лёгкий, с кучей расширений. Отлично подходит для начинающих.</li>
    <li><strong><a href="https://www.jetbrains.com/pycharm/download/?section=windows" target="_blank">PyCharm Community Edition</a></strong> — мощная IDE специально для Python.</li>
    <li><strong><a href="https://www.python.org/downloads/" target="_blank">IDLE</a></strong> — простейшая среда, которая устанавливается вместе с Python.</li>
    <li><strong><a href="https://jupyter.org/" target="_blank">Jupyter Notebook</a></strong> — для анализа данных и исследований.</li>
</ul>
<p>В следующих уроках мы подробно разберём установку и настройку самого популярного набора: Python + VS Code.</p>
