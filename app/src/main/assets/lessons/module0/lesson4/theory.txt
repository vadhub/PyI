<h2>Установка VS Code</h2>
<p>Скачайте VS Code с <a href="https://code.visualstudio.com/">официального сайта</a> и установите.</p>

<h3>Установка расширения для Python</h3>
<ol>
    <li>Запустите VS Code.</li>
    <li>Нажмите на значок расширений слева (или <code>Ctrl+Shift+X</code>).</li>
    <li>В поиске введите <code>Python</code> и выберите расширение от Microsoft (красный логотип). Установите его.</li>
    <li>Дополнительно можно установить <code>Pylance</code> и <code>Code Runner</code> (для быстрого запуска).</li>
</ol>

<h3>Открытие папки и создание файла</h3>
<ol>
    <li><code>File → Open Folder</code> — выберите вашу папку с проектами.</li>
    <li>Создайте новый файл (<code>File → New File</code>) и сохраните как <code>test.py</code>.</li>
    <li>Напишите код, например:
        <pre><code>print("Привет из VS Code!")</code></pre>
    </li>
    <li>Запустите, нажав зелёный треугольник в правом верхнем углу (или правой кнопкой мыши → Run Python File in Terminal).</li>
</ol>
<p>VS Code автоматически выберет интерпретатор Python, если он установлен. Результат появится в панели терминала внизу.</p>
