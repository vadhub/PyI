<h2>Шифр Цезаря</h2>
<p>Шифр Цезаря — один из простейших методов шифрования. Каждая буква в тексте заменяется на другую, стоящую на определённое число позиций дальше в алфавите.</p>
<p>Например, со сдвигом 3: A → D, B → E, C → F и т.д. В конце алфавита происходит циклический переход.</p>

<h3>Алгоритм</h3>
<ol>
  <li>Выбрать сдвиг (целое число).</li>
  <li>Для каждого символа текста:
    <ul>
      <li>Если это буква, найти её позицию в алфавите, добавить сдвиг, взять по модулю длины алфавита.</li>
      <li>Полученную позицию преобразовать обратно в букву.</li>
      <li>Если символ не буква, оставить без изменений.</li>
    </ul>
  </li>
</ol>

<p>Для упрощения можно работать только с английскими буквами (26).</p>
<p>Пример (сдвиг 3):</p>
<pre><code>text = "hello"
# Результат: "khoor"</code></pre>

<p>Дешифровка выполняется с отрицательным сдвигом.</p>
