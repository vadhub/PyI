<h2>Словари</h2>
<p>Словарь (dict) — это неупорядоченная структура данных, которая хранит пары "ключ — значение". Ключи должны быть уникальными и неизменяемыми (числа, строки, кортежи).</p>

<h3>Создание словаря</h3>
<pre><code># Пустой словарь
empty_dict = {}

# Словарь с данными
student = {
    "name": "Анна",
    "age": 20,
    "city": "Москва"
}

print(student)</code></pre>

<h3>Доступ к элементам</h3>
<pre><code>print(student["name"])   # Анна
print(student.get("age")) # 20 (безопасный способ, не вызывает ошибку при отсутствии ключа)</code></pre>

<h3>Изменение и добавление элементов</h3>
<pre><code>student["age"] = 21       # изменить существующее
student["email"] = "anna@example.com"  # добавить новое
print(student)</code></pre>

<p>Словари очень удобны для хранения структурированной информации.</p>
