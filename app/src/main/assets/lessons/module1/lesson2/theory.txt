Переменные и типы данных

В Python переменные используются для хранения данных.
Чтобы создать переменную, нужно просто придумать имя и присвоить ей значение с помощью знака "=".

Пример:
name = "Анна"
age = 25

Основные типы данных:
- int (целые числа): 5, -10, 1000
- float (числа с плавающей точкой): 3.14, -0.5, 2.0
- str (строки): "Привет", 'Python', "123"

Правила именования переменных:
- Имя может содержать буквы (a-z, A-Z), цифры и знак подчеркивания _
- Имя не может начинаться с цифры
- Имя не должно совпадать с ключевыми словами Python (if, else, while и т.д.)
- Регистр имеет значение: name и Name — разные переменные

Простые операции с числами:
- Сложение: 5 + 3 → 8
- Вычитание: 10 - 4 → 6
- Умножение: 3 * 7 → 21
- Деление: 15 / 3 → 5.0 (результат всегда float)
- Целочисленное деление: 15 // 2 → 7
- Остаток от деления: 15 % 2 → 1
- Возведение в степень: 2 ** 3 → 8

Строки можно складывать (конкатенация): "Hello" + " " + "World" → "Hello World"
Строку можно умножать на число: "Ha" * 3 → "HaHaHa"
