<h2>Переменные и типы данных</h2>
<p>В Python переменные используются для хранения данных.<br>
Чтобы создать переменную, нужно просто придумать имя и присвоить ей значение с помощью знака "=".</p>
<p>Пример:</p>
<pre><code>name = "Анна"
age = 25</code></pre>
<br>
<h2>Основные типы данных</h2>
<ul>
  <li><code>int</code> (целые числа): 5, -10, 1000</li>
  <li><code>float</code> (числа с плавающей точкой): 3.14, -0.5, 2.0</li>
  <li><code>str</code> (строки): "Привет", 'Python', "123"</li>
</ul>
<br>
<h2>Правила именования переменных</h2>
<ul>
  <li>Имя может содержать буквы (a-z, A-Z), цифры и знак подчеркивания _</li>
  <li>Имя не может начинаться с цифры</li>
  <li>Имя не должно совпадать с ключевыми словами Python (if, else, while и т.д.)</li>
  <li>Регистр имеет значение: name и Name — разные переменные</li>
</ul>
<br>
<h2>Простые операции с числами</h2>
<ul>
  <li>Сложение: <code>5 + 3</code> → 8</li>
  <li>Вычитание: <code>10 - 4</code> → 6</li>
  <li>Умножение: <code>3 * 7</code> → 21</li>
  <li>Деление: <code>15 / 3</code> → 5.0 (результат всегда float)</li>
  <li>Целочисленное деление: <code>15 // 2</code> → 7</li>
  <li>Остаток от деления: <code>15 % 2</code> → 1</li>
  <li>Возведение в степень: <code>2 ** 3</code> → 8</li>
</ul>
<br>
<p>Строки можно складывать (конкатенация): <code>"Hello" + " " + "World"</code> → "Hello World"<br><br>
Строку можно умножать на число: <code>"Ha" * 3</code> → "HaHaHa"</p>
