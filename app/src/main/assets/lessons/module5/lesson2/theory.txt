<h2>Аргументы и возврат значений</h2>
<p>Функции могут принимать входные данные (аргументы) и возвращать результат.</p>

<h3>Параметры и аргументы</h3>
<p>При определении функции в скобках указываются параметры — переменные, которые будут получать значения при вызове. Значения, передаваемые при вызове, называются аргументами.</p>
<pre><code>def greet(name):
    print("Привет,", name)

greet("Анна")  # Выведет: Привет, Анна</code></pre>

<h3>Возврат значения: return</h3>
<p>Инструкция <code>return</code> возвращает значение из функции и завершает её выполнение.</p>
<pre><code>def add(a, b):
    return a + b

result = add(5, 3)
print(result)  # 8</code></pre>

<p>Если <code>return</code> не указан, функция возвращает <code>None</code>.</p>
