<h2>Области видимости переменных</h2>
<p>Переменные, определённые внутри функции, называются локальными — они доступны только внутри этой функции. Переменные, определённые вне всех функций, называются глобальными — они доступны везде.</p>

<h3>Локальные переменные</h3>
<pre><code>def my_func():
    x = 10  # локальная переменная
    print(x)

my_func()
# print(x)  # Ошибка: x не определена вне функции</code></pre>

<h3>Глобальные переменные</h3>
<pre><code>y = 5  # глобальная переменная

def show():
    print(y)  # можно читать глобальную

show()
print(y)      # и здесь доступна</code></pre>

<h3>Изменение глобальных переменных внутри функции</h3>
<p>Чтобы изменить глобальную переменную, нужно использовать ключевое слово <code>global</code>. Обычно этого лучше избегать, но для понимания механизма полезно знать.</p>
<pre><code>counter = 0

def increment():
    global counter
    counter += 1

increment()
increment()
print(counter)  # 2</code></pre>
