<h2>Списки</h2>
<p>Список (list) — это упорядоченный набор элементов, которые могут быть разных типов. Списки очень гибкие и часто используются в Python.</p>

<h3>Создание списка</h3>
<p>Список создаётся с помощью квадратных скобок <code>[]</code>, элементы перечисляются через запятую.</p>
<pre><code># Пустой список
empty_list = []

# Список чисел
numbers = [1, 2, 3, 4, 5]

# Список строк
fruits = ["яблоко", "банан", "апельсин"]

# Смешанный список
mixed = [1, "hello", 3.14, True]</code></pre>

<h3>Индексация</h3>
<p>Каждый элемент имеет индекс (порядковый номер). Индексация начинается с <code>0</code>. Можно использовать отрицательные индексы для доступа к элементам с конца.</p>
<pre><code>fruits = ["яблоко", "банан", "апельсин"]
print(fruits[0])   # яблоко
print(fruits[1])   # банан
print(fruits[-1])  # апельсин (последний)
print(fruits[-2])  # банан (предпоследний)</code></pre>

<h3>Изменение элементов</h3>
<pre><code>fruits[1] = "киви"
print(fruits)  # ["яблоко", "киви", "апельсин"]</code></pre>
